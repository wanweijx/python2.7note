# 内建函数
##　Python提供了很多的内建函数
* 1 数学运算函数<br>
    abs()     // 取绝对值函数  abs(-5)=5<br>
    round()   // 四舍五入函数  round(2.6)=3.0<br>
    pow()     // pow(2,2)=4  2的2次方  pow(2,2,3)=4%3<br>
    cmp()     // 比较两个值的大小 cmp(num1,num2)  num1大则返回1 num1小则返回-1<br>
    divmod()  // 返回除法结果的值和余数  求余算法 divmod(10,4)  (2,2)<br>
    max()     // 取最大值 返回最大值<br>
    min()     // 取最小值 返回最小值<br>
    sum()     // 求和<br>
